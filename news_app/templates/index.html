서비스 제작 과정

  • 뉴스 크롤링

    네이버에서 제공되는 '세계' 카테고리의 헤드라인 기사를 selenium 사용하여 스크래핑하였습니다. renew 버튼을 사용하여 사용자가 직접 뉴스를 갱신할 수 있습니다.
    Crawling URL

  • 한국어 요약 모델 생성

    PageRank는 구글 검색 알고리즘의 기초를 마련했습니다. 이를 text에 적용시킨 방법인 TextRank을 활용하여 한국어 요약 모델을 생성하였습니다.
    Reference

  • 웹 제작

    bootstrap 테마를 활용하여 template을 구성하고 Flask를 통해 웹을 동작시켰습니다. 클라우드 데이터베이스인 PostgreSQL을 사용하여 뉴스 기사 데이터를 관리하였습니다.
    Bootstrap theme URL

  • 웹서비스

    사용자는 renew 버튼을 통해 현재 날짜의 뉴스들을 자동으로 수집할 수 있습니다. 수집된 뉴스들은 자동으로 요약되어 데이터베이스에 저장되게 됩니다. 데이터베이스에 저장되어 있는 헤드라인 뉴스 요약본들은 사용자가 날짜를 선택함으로써 불러올 수 있습니다.
